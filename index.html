📖 JSONBin 使用教學

5 分鐘設定，之後手機/電腦自動同步。

  1. 打開 https://jsonbin.io，右上 Sign Up 註冊（建議用 Google 登入）。
  2. 登入後左側選單點 API Keys。
  3. 找到 X-MASTER-KEY 那一欄，點右邊複製按鈕。
  4. 回到本面板，把 Master Key 貼到上方輸入框。
  5. 按 ✨ 建立新 Bin 按鈕 → 系統會自動建立並填入 Bin ID。
  6. 完成！之後手機/電腦只要貼上同一組 Master Key + Bin ID，就會看到同一份卡片。
📱 換到手機怎麼用？
手機打開同一個網址 → 展開「⚙️ 雲端設定」→ 貼上同一個 Master Key 跟 Bin ID → 按「⬇️ 立即從雲端同步」就會看到電腦的卡片。
